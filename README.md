# 📊 Mobile Sales Dashboard (Power BI)

## 🚀 Overview
This project presents an interactive **Mobile Sales Dashboard** built using Power BI to analyze sales performance, customer behavior, and business trends. It enables data-driven decision-making through dynamic visualizations and KPI tracking.

---

## 📌 Key Features
- 📈 **KPI Metrics**: Total Sales, Quantity Sold, Transactions
- 📅 **Time Analysis**: Yearly, Quarterly, Monthly, and Daily trends
- 🔍 **Dynamic Filters**: Mobile Model, Payment Method, Brand, Time Period
- 📊 **Comparative Analysis**: Same Period Last Year vs Current Sales
- 🌍 **Geographical Insights**: Sales distribution across cities
- 💳 **Payment Analysis**: UPI, Debit Card, Credit Card, Cash breakdown
- ⭐ **Customer Insights**: Ratings and transaction behavior

---

## 🛠️ Tools & Technologies
- **Power BI**

- **Data Modeling**
